Activity

A list of dates with associated descriptions for that date, the date has the class .activity-list__date and the description has the class .activity-list__description.

  • 14th March 2016
  • event for 14th March 2016

  <li class="[modifier class]">14th March 2016</li>
  <li class="activity-list__description">event for 14th March 2016</li>
</ul>

.list--collapse

A collapsable list element on smaller breakpoints. Inline list at larger breakpoints. List items have 'list__item' class and toggle item has attribute 'data-list--collapse-toggle'

  • I'm the toggle
  • list item link
  • another list item link 
<ul class="list--collapse hide-inline-list--collapse-targets">
    <li class="list__item" data-list--collapse-toggle>
        <span>I'm the toggle</span>
    </li>
    <li class="list__item">
        <a href="#">list item link</a>
    </li>
    <li class="list__item">
        <a href="#">another list item link</a>
    </li>
</ul>

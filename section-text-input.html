Text Input

Discuss this component on Hackpad

Text Inputs are used for obtaining information from users in a form.

<input name="example_text_input" type="text" class="[modifier class]" value="example value" />

.input--xxsmall

An extra extra! small input 
<input name="example_text_input" type="text" class="input--xxsmall" value="example value" />

.input--xsmall

An extra small input 
<input name="example_text_input" type="text" class="input--xsmall" value="example value" />

.input--small

A small input 
<input name="example_text_input" type="text" class="input--small" value="example value" />

.input--normal

A normal input 
<input name="example_text_input" type="text" class="input--normal" value="example value" />

.input--fullwidth

A full width input 
<input name="example_text_input" type="text" class="input--fullwidth" value="example value" />

.input--no-spinner

An input without a spinner 
<input name="example_text_input" type="text" class="input--no-spinner" value="example value" />

.input--cleared

To make an input {display: block} 
<input name="example_text_input" type="text" class="input--cleared" value="example value" />

.input--link

To make an input appear like a link 
<input name="example_text_input" type="text" class="input--link" value="example value" />

.input--left-padding

An input with left padding giving space for symbols 
<input name="example_text_input" type="text" class="input--left-padding" value="example value" />

Currency

Currency inputs add the £ symbol visually within the input field

<span class="input-currency">
  <input name="example_monetary_input" type="number" class="input--small input--no-spinner input--left-padding" value="0.99" />
</span>

Number

Number Inputs are used for entering a floating point number.

<input name="example_number_input" type="number" class="input--xxsmall input--no-spinner" value="example value" />

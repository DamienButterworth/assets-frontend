Label

Discuss this component on Hackpad

Labels are used to describe an input or group of inputs.

<label class="[modifier class]">
  First Name
  <input class="input--cleared" name="example" type="text" value="example value" />
</label>

.indent

An indented label 
<label class="indent">
  First Name
  <input class="input--cleared" name="example" type="text" value="example value" />
</label>

.bold

A bold label 
<label class="bold">
  First Name
  <input class="input--cleared" name="example" type="text" value="example value" />
</label>

.label--inline

An inline label 
<label class="label--inline">
  First Name
  <input class="input--cleared" name="example" type="text" value="example value" />
</label>

A tight inline label

</label>

.label--inline-right

A right aligned inline label

  First Name
  <input class="input--cleared" name="example" type="text" value="example value" />
</label>

Form date

A container element using the .form-date css selector will change how a child label appears.

<div class="form-date">
  <label>
    <input class="input--xxsmall input--no-spinner" name="example" type="number" />
    Yellow
  </label>
  <label>
    <input class="input--xxsmall input--no-spinner" name="example" type="number" />
    Orange
  </label>
</div>
